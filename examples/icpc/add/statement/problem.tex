\problemname{A + B Problem}

Given two numbers $a$ and $b$, please calculate $a+b$.

\subsection*{Input}

The input only contains one line with two integers $a$, $b$.

\begin{itemize}
    \item $1 \leq a, b < 2^{31}$
\end{itemize}

\subsection*{Output}

Output a single line with sum of $a$ and $b$.
